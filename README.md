## Repository Structure

```text
letkod/
├── solutions/        # Python solutions (source of truth)
├── scripts/          # Automation scripts
├── .gitignore
└── README.md
```

## Project Structure

*   `solutions/`: Contains the Python solution files for each LeetCode problem.
*   `scripts/`: Contains utility scripts, including the script to synchronize all documentation. 
*   `README.md`: This file, which is auto-generated by a script.

## Usage

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/kaylaradf/letkod.git
    cd letkod
    ```
2.  **Navigate to a solution:**
    ```bash
    cd solutions
    ```
3.  **Run a solution (example):**
    ```bash
    python 1.two-sum.py
    ```
